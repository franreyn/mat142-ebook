Exponential Models

India is the second most populous country in the world with a population of about 1.39 billion people in 2021. The population is growing at a rate of about 1.0% each year (Source: worldpopulationreview.com accessed July 2021). If this rate continues, the population of India will exceed China’s population by the year 2030. When populations grow rapidly, we often say that the growth is “exponential,” meaning that something is growing very rapidly. To a mathematician, however, the term exponential growth has a very specific meaning. In this section, we will take a look at exponential models, which model this kind of rapid growth.

Identify elements of an exponential model (equation)

When exploring linear growth, we observed a constant rate of change—a constant number by which the output increased for each unit increase in input. For example, in the equation y = 3 x + 4 , the slope tells us the output increases by 3 each time the input increases by 1. The scenario in the India population example is different because we have a percent change per unit time (rather than a constant change) in the number of people.

Exponential Model

An exponential model (equation) has a form y = a ⋅ b x , where a is any nonzero number, b is a positive real number not equal to 1.

  • If b > 1 , we say it models exponential growth. As x increases, the outputs of the model will increase as well and continue to increase at a much faster rate without bound.
  • If 0 < b < 1 , the model shows exponential decay. As x increases, the outputs for the model decrease and continue to decrease, but levels off to the x -axis (gets close to zero, but will not equal to zero).
  • When x = 0 the output is a , so we often call a the initial amount in applications (or the starting amount). We also say a is also the y-intercept on the graph for the model.
  • The value b is what we call the growth multiplier. If we increase x by 1, then b is what we multiply the previous output value by to get the new output.

For us to gain a clear understanding of exponential growth, let us contrast exponential growth with linear growth. We will construct two models. The first model is exponential. We will start with an input of 0, and increase each input by 1. The second model is linear. For both models they will start with the same outputs for on the first two steps. See the table below

Example 1

Which of the following equations are examples of exponential models? If it is an exponential model identify the starting amount (y-intercept value) and the growth multiplier.

  1. y = 4 x
  2. y = 2 x 4
  3. y = 5 ⋅ 4 x
  4. y = 3 ⋅ ( 1 2 ) x

Solution

  1. Not Exponential. This is a linear model.
  2. Not Exponential. The base for y = 2 x 4 is a variable and not a constant.
  3. Exponential model. Starting amount is 5 and growth multiplier is 4.
  4. Exponential model. Starting amount is 3 and growth multiplier is 1 2

Evaluate an exponential model

To evaluate an exponential model with the form y = b x , we substitute x with the given value and calculate the resulting power. For example:

Let y = 3 x . What is the value of the output for the model when x = 2 ?

y = 3 x y = 3 2  Substitute   x = 2 y = 9    Evaluate the power

To evaluate an exponential model with a form y = a ⋅ b x it is important to follow the order of operations. For example:

Let y = 30 ⋅ 2 x . Whatis the value of the output for the model when x = 3 ?

y = 30 ⋅ 2 x y = 30 ⋅ 2 3   Substitute  x = 3 y = 30 ⋅ 8    Simplify the base to a power first y = 240    Multiply

It is important to follow the order of operations as you will get an incorrect result by not doing so:

y = 30 ⋅ 2 3 ≠ 60 3 = 216 , 000

Example 2

Evaluate the exponential model y = 5 ( 1 2 ) x at x = 2 and x = 5

Solution

Evaluate at x = 2 . Make sure to follow the order of operations:

y = 5 ⋅ ( 1 2 ) x y = 5 ⋅ ( 1 2 ) 2 y = 5 ⋅ ( 1 4 ) y = 5 4

Evaluate at x = 5 .

y = 5 ⋅ ( 1 2 ) x y = 5 ⋅ ( 1 2 ) 5 y = 5 ⋅ ( 1 32 ) y = 5 32

Create an exponential model with algebra between two quantitative variables

Before we explore applications and the graph of an exponential model let us first look at strategies for finding the exponential equation from two points. We will only consider applications where both points have positive y-values (both are above the x-axis when plotted). Below are a few algebra facts we will need to know for the process.

Properties of Exponents

These properties of exponents are leveraged in this section, especially the quotient rule, which you will use immediately below to solve for the base of an exponential model when given two points.

Properties of Exponents: For a ≠ 0 and b ≠ 0 where m and n are postive integers.
Property Definition Example

Property of 1

b 1 = b

5 1 = 5

Property of 0

b 0 = 1

3 0 = 1

Product Property

( a m ) ( a n ) = a ( m + n )

( 2 3 ) ( 2 5 ) = 2 ( 3 + 5 ) = 2 8

Property of Negatives

b − n = 1 b n

2 − 3 = 1 2 3 = 1 8

Quotient Property

a m a n = a ( m − n )

2 4 2 3 = 2 ( 4 − 3 ) = 2 1 = 2

There are many more exponent properties in algebra, but only the above is needed for our work with exponential growth.

The other facts from algebra we will need are given below about solving equations where the unknown variable is in the base and it is raised to an integer power along with a general strategy for solving for an exponential model.

Solving equation of the form c = x n

Let c be a positive number, n be a nonzero integer, and x > 0 . To solve an equation of the form c = x n you raise both sides of the equation to the 1 n power.

c = x n c ( 1 n ) = ( x n ) ( 1 n ) c 1 n = x

General Strategy for find Exponential Model of the form y = a b x given two points ( x 1 , y 1 )  and  ( x 2 , y 2 )

  • If one of the points is a y-intercept, i.e. ( 0 , a ) , then the initial amount a is already given to you. Using a , substitute the other point into the exponential model y = a ⋅ b x and solve for b .
  • If neither points given has the form ( 0 , a ) you can solve for a and b by substituting both points into the exponential model to create two equations. Solve the resulting system of equations.

The above assumes both y 1 ⋅ y 2 > 0  and  x 1 ⋅ x 2 ≠ 0 (this means both points are above the horizontal axis or below the horizontal axis).

Let us look now at a few examples of creating an exponential model. In the next example below we will be given two values where one is the y-intercept.

Example 3

Given two points ( 0 , 3 ) and ( 3 , 3 8 )

Solution

Start with the exponential model y = a b x and use the point that gives the initial amount, ( 0 , 3 ) . At this point we can say a = 3 , but below shows why that is true showing what happens when we put the values in the point into the model.

3 = a b 0 3 = a ⋅ 1 3 = a

We now solve for b in the model ( 3 , 3 8 ) , using the point.

y = 3 b x 3 8 = 3 b 3 1 3 ⋅ 3 8 = 1 3 ⋅ 3 b 3 1 8 = b 3 ( 1 8 ) 1 3 = b 1 2 = b

The exponential model is y = 3 ( 1 2 ) x

Evaluating roots on the calculator

In the previous example, we had to calculate the 3rd root of a number. This is different than taking the basic square root, √. Many scientific calculators have a button for general roots.  It is typically labeled like:

n ,   x ,   or  x y

To evaluate the 3rd root of 8, for example, we’d either type 3  n  8, or 8  n  3, depending on the calculator.  Try it on yours to see which to use – you should get an answer of 2.

If your calculator does not have a general root button, all is not lost.  You can instead use the property of exponents which states that a n = a 1 / n .  So, to compute the 3rd root of 8, you could use your calculator’s exponent key to evaluate 8 1 / 3 .  To do this, type:
8 y x ( 1 ÷ 3 )  or  8 ^ ( 1 ÷ 3 )

The parentheses tell the calculator to divide 1/3 before doing the exponent.

In the next example we will find the exponential model for two general points where neither is the y-intercept.

Example 4

Given two points ( − 2 , 6 )  and  ( 2 , 1 ) find the exponential model that goes through them. Round the values of a and b to four decimal places.

Solution

Because we don’t have the initial value, we substitute both points into an equation of the form y = a ⋅ b x , and then solve the system for a and b .

  • Substituting ( − 2 , 6 ) gives 6 = a ⋅ b − 2
  • Substituting ( 2 , 1 ) gives 1 = a ⋅ b 2

Use the first equation to solve for a in terms of b :

6 = a ⋅ b − 2 6 b − 2 = a Divide a = 6 b 2 Rewrite the denominator with properties of exponents

Substitute a in the second equation, and solve for b :

1 = a ⋅ b 2 1 = ( 6 b 2 ) b 2 Substitute in  a = 6 b 2 1 = 6 b 4 Simplify using property of exponents 1 6 = b 4 Divide both sides by 6 ( 1 6 ) 1 4 = b Use property of exponents to isolate  b b ≈ .6389

Use the value of b in the first equation to solve for the value of a :

a = 6 b 2 a ≈ 6 ( 0.6389 ) 2 a ≈ 2.4492

The equation for the exponential model is: y = 2.4492 ⋅ ( 0.6389 ) x

Graph an exponential model

Earlier we compared exponential growth to linear growth from a table of values. Let us revisit the linear and exponential model, but instead compare their graphs to point out some characteristics of an exponential models graph.

When graphing linear models we said only two points are needed. The same can be said about an exponential model of the form y = a ⋅ b x , but the preference for the points to use would be the y-intercept at ( 0 , a ) and point where the input is 1, ( 1 , a b ) . The reason for the choice of the second point is that for each one unit increase in x we will multiply the y coordinate by b . Going the other direction for each one unit decrease in x we will divide the y coordinate by b .

Let us examine the graph of y = 2 x by plotting the ordered pairs.

Some observations we can make is that if you look at the point ( 1 , 2 ) we can move forward and backwards with one unit in x by multiplying or dividing by the base 2. Moving forward we multiply by the base and moving backwards we divide by the base.

We also see that as x increases that this exponential model will increase in the output values and starts to grow rapidly (the changes in output get larger with each step). As x decreases (gets more negative) the graph is approaching the horizontal axis.

Solve application problems using an exponential model created with algebra

At the start of this section we referenced the population of India growing not by a fixed amount each year, but rather a fixed percent increase. Situations where a quantity is growing by a fixed percentage for each unit increase is one that can be modeled with an exponential equation of the form y = a ⋅ b x .

Find an exponential model in applications

  1. Identify the variables in the given information
  2. Assign the variables letter names
  3. If the initial amount is given in the information assign that value to a in the model. Using the value of a substitute the other information into the exponential model and solve b .
  4. If the initial amount is not given you can create two equations and solve this system for a and b .

Example 6

The population in Tucson in 2019 was approximately 548.1 thousand people and 551.3 thousand people in 2020. If the population continues to grow by the same percent each year (grows exponentially), then what would be the population in the year 2030?

Solution

The information provided can be used to build an exponential model as we have two data points to work with. The first is an initial amount of the population in the year 2019 and the second is the population one year later. To build the model we will put it in the form P = a ⋅ b t , where t represents the number of years since 2019 and P is the population.

In the model P = a ⋅ b t we know a represents the initial amount. This means a = 548.1 as that is the population in the year 2019 (0 years since 2019).

To find b can put in the second piece of information into the model. The population in 2020 was 551.3 thousand, since 2020 is 1 year after 2019 we know when t = 1 that P = 551.3

P = 548.1 ⋅ b t 551.3 = 548.1 ⋅ b 1 551.3 548.1 = b 1.005838 = b

We can write the model as P = 548.1 ⋅ ( 1.005838 ) t

Note: we rounded the value of b to six decimal places to reduce the amount of rounding errors in the work we do afterwards with the model. It is best to use the exact value whenever possible, but it can lead to formulas looking more complex then need be.

To find the population in 2025 we evaluate the model at t = 2025 − 2019 = 6 .

P = 548.1 ⋅ ( 1.005838 ) 6 P ≈ 548.1 ⋅ 1.035543 P ≈ 567.6

The population is estimated to be 567.6 thousand people in the year 2025 based on our exponential model.

Why did we choose in the example above for t to be the number of years since 2019 and not just the year itself? Part of the reason for this was to cut down the work needed to create the model. By using 2019 as the initial year we were able to insert that population in for the value of a. If we had not done that we would have had to solve a system of equation of the form below:

548.1 = a ⋅ b 2019 551.3 = a ⋅ b 2020

Solve for a and b would result in the same value for b as we had above, but a ≈ 0.0043097 . This would give us a model of the form P = 0.0043097 ⋅ ( 1.005838 ) t . Looking at this model we lose some connection back to the application as the initial amount now represents the population in year 0. The version where we allow t to be the number of years since 2019 allows us to connect the information in the model with the population for the year 2019 (it retains more information for the reader as it gives them the population in 2019 for the initial amount).

Example 7

The number of employees at Amazon has been growing at an exponential rate. Find a model for the number of employees since 2006 given that there were 88,400 employees in 2012 and 154,100 in 2014. Using this model how many employees would be at amazon in 2020?

Solution

Let N be the number of employees at Amazon and t be the number of years after 2006. The model would have the form N = a ⋅ b t . 2006 represents the starting year for the model, so we are not given any initial amount as the first amount known is for the year 2008 ( t = 2 ) and the number of employees as 88,400. The given information can be written as two ordered pairs: ( 2 ,   88400 ) and ( 6 ,   154100 ) .

Create two equations based on the given points

  • Substituting ( 2 ,   88400 ) gives 88400 = a ⋅ b 2
  • Substituting ( 6 ,   154100 ) gives 154100 = a ⋅ b 6

Use the first equation and solve for a in terms of b :

88400 = a ⋅ b 2 88400 b 2 = a

Substitute a in the second equation and solve for b :

154100 = a ⋅ b 6 154100 = ( 88400 b 2 ) ⋅ b 6 154100 = 88400 ⋅ b 6 b 2 154100 = 88400 ⋅ b 6 − 2 154100 = 88400 ⋅ b 4 154100 88400 = b 4 ( 154100 88400 ) 1 4 = b 1.149046 ≈ b

Use the value of b in the first equation to solve for the value of a :

88400 = a ⋅ b 2 88400 = a ⋅ ( 1.149046 ) 2 88400 ( 1.149046 ) 2 = a 66954 ≈ a

The equation for the exponential model is N = 66954 ⋅ ( 1.149046 ) t

Use the model to predict the number of employees in 2020. Use t = 14 in the model:

N = 66954 ⋅ ( 1.149046 ) 14 N ≈ 468274

The model predicts that in 2020 Amazon will have approximately 468274 employees.

  16. The population of a small town is modeled by P = 45 , 000 ⋅ ( 1.035 ) t , where t is the number of years after 2010. What was the population in 2010? Is the population increasing or decreasing?
    Answer

    The population in 2010 is 45,000. The population is growing as the base is greater than 1.

  17. Suppose that you have a bowl of 500 Skittles, and each day you eat 1 4 of the Skittles you have. Is the number of candies left changing linearly or exponentially? Write an equation to model the number of candies left after t days.
    Answer

    The number of candies left would be changing exponentially. The model would be N = 500 ⋅ ( 0.75 ) t
